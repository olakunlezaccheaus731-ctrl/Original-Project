# Cowrywise Landing Page

A modern, responsive landing page for Cowrywise - a fintech platform for savings and investment management.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **User Authentication**: Login and signup pages with form validation
- **Interactive UI**: Smooth animations and transitions
- **Professional Testimonials**: Customer success stories with multimedia cards
- **Product Showcase**: Detailed sections for different investment products
- **Security**: SEC regulated fund manager with transparent information

## Pages

- `index.html` - Main landing page with hero section, features, and testimonials
- `login.html` - User login page with email/password authentication
- `signup.html` - Account creation page with email verification

## Project Structure

```
├── index.html          # Main landing page
├── login.html          # Login page
├── signup.html         # Sign up page
├── images/             # Image assets
├── .gitignore          # Git ignore rules
└── README.md           # This file
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Navigate through the pages using the navigation menu
